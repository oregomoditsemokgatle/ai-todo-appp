# AI To-Do App

This is a simple AI-style to-do list application built with HTML, CSS, and JavaScript.

## Features

- Add tasks
- Delete tasks
- Mark tasks as complete
- Save tasks using localStorage
- Generate AI-style task suggestions

## Why I Built This

I built this project to practise front-end development skills and show that I can create interactive web applications. I also wanted to include an AI-inspired feature that suggests tasks for the user.

## What I Learned

- DOM manipulation
- JavaScript functions
- Event handling
- Local storage
- Building clean user interfaces
- Debugging small application errors
